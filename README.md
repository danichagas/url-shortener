# API de Encurtador de URLs

Uma API REST de encurtador de URLs robusta e profissional desenvolvida com **NestJS**, **TypeScript**, **Docker** e **MongoDB**.

## Funcionalidades
- Encurtar URLs longas gerando um código aleatório exclusivo de 6 caracteres.
- Redirecionar os utilizadores da URL curta para o destino original.
- Arquitetura profissional utilizando módulos do NestJS e Mongoose.
- Banco de dados MongoDB totalmente containerizado com Docker Compose.

## Tecnologias Utilizadas
- **Backend:** Node.js, NestJS, TypeScript
- **Banco de Dados:** MongoDB (via Mongoose)
- **Containerização:** Docker & Docker Compose
- **Bibliotecas:** nanoid
